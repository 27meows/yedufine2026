# 얘두파인?

교사를 위한 AI 업무 문장 다듬기 서비스

## 프로젝트 소개

얘두파인은 학교 현장에서 자주 작성하는 학부모 안내, 교직원 공지, 메신저, 업무 안내 등의 문장을 상황에 맞게 자연스럽게 다듬어주는 AI 웹서비스입니다.

OpenAI API를 활용하여 상황, 대상, 말투를 반영한 문장을 생성합니다.

## 주요 기능

- AI 문장 진단
- 추천 문장 3개 생성
- 수정 포인트 제공
- 추천 문장 복사
- 상황/대상/말투 선택
- 선택한 추천 문장 AI 재수정

## 기술 스택

- HTML
- CSS
- JavaScript
- OpenAI API

## 실행 방법

1. 웹사이트에 접속합니다.
2. OpenAI API Key를 입력합니다.
3. 원문, 상황, 대상, 말투를 선택합니다.
4. `얘두파인?` 버튼을 누르면 AI가 문장을 분석하고 추천 문장을 생성합니다.

- 수업 영상 욕설·비속어 검토
- 조퇴 가능일 자동 계산
- 공문 캡처 업로드 및 문서 점검
- 교사용 AI 업무 도우미 기능 확장
